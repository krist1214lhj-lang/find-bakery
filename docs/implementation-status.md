## 알려진 제한

- 조회 경로는 실제 Supabase를 사용하지만 현재 레코드 내용은 UI와 로직 검증용 가상 시드다.
- 로컬 화면 실행에는 `.env.local`의 `NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_ANON_KEY`가 필요하다.
- 기본 영업시간은 아직 단일 일일 영업 구간만 사용한다.
- 저장은 기기 로컬 저장소 기반이며 계정 간 동기화되지 않는다.
- 서버 장애 시 로컬 폴백 제보는 해당 기기에만 보관되며 자동 재전송은 아직 없다.
- 원격 Supabase 테스트 프로젝트에는 아직 연결하지 않았다.
- 후보는 사용자 요청과 관리자 승인 없이는 정식 빵집으로 공개되지 않는다.
- 후보 저장→관리자 승인 실제 브라우저 통합 검증은 실행 도구 승인 한도로 인해 대기 중이다.
- 공개 제보 API의 운영용 지속형 속도 제한은 아직 연결하지 않았다.
- `npm run verify:deploy-env`는 현재 Supabase·배포 환경 변수가 없어 의도대로 실패하며 배포를 차단한다.
- 작업명 `빵길`은 공개 검색 1차 확인만 했으며 출시 전 상표·스토어 정식 확인이 필요하다.
