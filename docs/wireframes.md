# 핵심 화면 와이어프레임

작성 기준일: 2026-06-18  
기준 화면: 모바일 390×844px

## 1. 홈

모바일 홈은 아래 네 장면을 각각 390×844px 한 화면으로 구분하고 세로 스냅으로 이동한다.

1. 브랜드 영상·검색
2. 먹고 싶은 빵
3. 최근 확인된 빵집 — 카드는 가로 스와이프
4. 지역별 빵길

```text
┌─────────────────────────────────┐
│ 빵길                      [프로필]│
│ 믿고 떠나는 전국 빵집 지도       │
│                                 │
│ 빵 냄새를 따라,                  │
│ 확실한 길로.                     │
│ ┌─────────────────────────────┐ │
│ │ 세로형 브랜드 영상           │ │
│ │ 오늘의 빵길 · 갓 구운 곳으로 │ │
│ └─────────────────────────────┘ │
│ [출발표 카드]       [확인 스탬프]│
│ ┌─────────────────────────────┐ │
│ │ 🔍 지역, 빵집, 빵 이름 검색  │ │
│ └─────────────────────────────┘ │
│ 출처 ✓  확인일 ✓  변경 제보 ✓   │
│                                 │
│ 오늘 마음이 가는 빵은?           │
│ [소금빵] [베이글] [크루아상]      │
│ [식사빵] [케이크] [구움과자]      │
│                                 │
│ 지금 정보가 확인된 빵집      전체 │
│ ┌──────────────┐ ┌────────────┐ │
│ │ 사진         │ │ 사진       │ │
│ │ 빵집명       │ │ 빵집명     │ │
│ │ 서울 · 소금빵│ │ 대전 · 전통│ │
│ │ ✓ 7일 전 확인│ │ ✓ 오늘 확인│ │
│ └──────────────┘ └────────────┘ │
│                                 │
│ 지역별 빵길                      │
│ [서울] [부산] [대전] [대구] ...  │
│                                 │
│ 이번 주 테마                     │
│ ┌─────────────────────────────┐ │
│ │ 기차 타고 떠나는 빵지순례    │ │
│ │ 선정 근거와 8개 지점          │ │
│ └─────────────────────────────┘ │
│                                 │
│ [홈]          [탐색]       [저장]│
└─────────────────────────────────┘
```

상태:

- 로딩: 카드 스켈레톤, 검색과 카테고리는 즉시 사용 가능
- 데이터 없음: 추천 영역만 숨기고 탐색 진입 유지
- 오류: 전체 화면을 막지 않고 해당 섹션에서 재시도

## 2. 탐색 — 목록

```text
┌─────────────────────────────────┐
│ ← [서울 소금빵 검색________]     │
│ [지역] [빵 종류] [영업 중] [더] │
│                                 │
│ 24곳                  [지도 보기]│
│ ┌─────────────────────────────┐ │
│ │ [사진]  빵집명 성수점      ♡ │ │
│ │         소금빵 · 크루아상    │ │
│ │         영업 중 · 19:00 종료 │ │
│ │         A 공식 확인 · 2일 전 │ │
│ │         1.2km                │ │
│ └─────────────────────────────┘ │
│ ┌─────────────────────────────┐ │
│ │ [사진]  빵집명 연남점      ♡ │ │
│ │         정보 확인 필요       │ │
│ │         C · 112일 전 확인    │ │
│ └─────────────────────────────┘ │
│                                 │
│ [홈]          [탐색]       [저장]│
└─────────────────────────────────┘
```

정렬 기본값:

1. 검색 관련도
2. 검증 상태
3. 거리 또는 지역 대표성

별점이 없어도 결과를 설명할 수 있어야 한다.

## 3. 탐색 — 지도

```text
┌─────────────────────────────────┐
│ ← [서울 소금빵 검색________]     │
│ [지역] [빵 종류] [영업 중] [더] │
│─────────────────────────────────│
│                                 │
│       ●A             ●B         │
│               ●A                │
│    ●C                           │
│                         ◎ 내 위치│
│                                 │
│                        [목록 24] │
│ ┌─────────────────────────────┐ │
│ │ [사진] 빵집명 · 영업 중      │ │
│ │ 소금빵 3,500원 · A · 2일 전  │ │
│ │ [상세 보기]        [저장 ♡]  │ │
│ └─────────────────────────────┘ │
└─────────────────────────────────┘
```

- 마커 문자는 검증 등급이 아니라 클러스터/선택 상태를 우선 표현한다.
- 등급은 미리보기 카드에서 텍스트로 제공한다.
- 지도 이동 후 `이 지역 검색`을 명시적으로 눌러 결과를 갱신한다.

## 4. 빵집 상세

```text
┌─────────────────────────────────┐
│ ←                         공유 ♡ │
│ [대표 이미지 16:10              ]│
│                                 │
│ 빵집명 성수점                    │
│ 서울 성동구 ○○로 12              │
│ [영업 중] 19:00 종료             │
│                                 │
│ ┌─────────────────────────────┐ │
│ │ ✓ 핵심 정보 공식 확인 A      │ │
│ │ 2026.06.16 확인              │ │
│ │ [출처와 확인 기록 보기]       │ │
│ └─────────────────────────────┘ │
│                                 │
│ [길찾기]       [전화]       [저장]│
│                                 │
│ 꼭 먹어볼 빵                     │
│ ┌────────┐ ┌────────┐          │
│ │ 사진   │ │ 사진   │          │
│ │ 소금빵 │ │ 크루아상│          │
│ │ 3,500원│ │ 4,800원│          │
│ │ 6/10확인│ │ 6/10확인│         │
│ └────────┘ └────────┘          │
│                                 │
│ 오늘의 영업 정보                 │
│ 수요일 10:00–19:00              │
│ ⚠ 재료 소진 시 조기 마감 가능    │
│ [전체 영업시간]                  │
│                                 │
│ 편의 정보                        │
│ 포장 가능 · 좌석 적음 · 주차 없음 │
│                                 │
│ 유명한 이유                      │
│ 지역 대표 제과점 · ○○상 수상     │
│ 근거 보기                        │
│                                 │
│ 정보가 다른가요? [수정 제보]     │
└─────────────────────────────────┘
```

상세 상단의 등급은 핵심 필드의 최저 등급을 무조건 대표값으로 쓰지 않는다. `영업 정보 A`, `메뉴 가격 C`처럼 필요한 경우 항목별 상태를 보여준다.

```text
┌─────────────────────────────────┐
```

```text
│                                 │
│ 어떤 정보가 다른가요?            │
│ ( ) 영업시간·휴무                │
│ ( ) 폐점 또는 이전               │
│ ( ) 메뉴·가격                    │
│ ( ) 주소·전화번호                │
│ ( ) 기타                         │
│                                 │
│ 확인한 내용을 알려주세요          │
│ ┌─────────────────────────────┐ │
│ │                             │ │
│ └─────────────────────────────┘ │
│                                 │
│ 출처 링크 (선택)                 │
│ [https://____________________]  │
│                                 │
```

제출 결과:

- 성공: 접수 번호와 검수 대기 안내
- 중복 가능: 기존 제보가 있음을 알리되 제출 선택 허용
- 실패: 입력값 유지 후 재시도

## 7. 관리자 검수

```text
┌────────────────────────────────────────────────────┐
│ 검수 대기 12건       [유형] [지역] [오래된 순]       │
├──────────────┬─────────────────────────────────────┤
│ 제보 목록     │ 빵집명 성수점 · 영업시간 변경        │
│ > 성수점      │                                     │
│   본점        │ 현재 값       제안 값                 │
│   부산점      │ 10:00–19:00   11:00–20:00            │
│              │                                     │
│              │ 제보 설명 / 원문 링크                 │
│              │ 공식 SNS: 6/18 게시, 6/20부터 적용    │
│              │                                     │
│              │ 변경 종류                            │
│              │ (●) 상시 변경  ( ) 임시 일정          │
│              │                                     │
│              │ [보류] [반려]             [승인]      │
└──────────────┴─────────────────────────────────────┘
```

승인 시 데이터 변경, 검증 기록, 검수 이력이 하나의 트랜잭션으로 처리되어야 한다.

## 8. 재검증 대기열

```text
┌─────────────────────────────────────────┐
│ 정보 재검증 대기열                       │
│ [출처 충돌 1] [기한 경과 4] [곧 재검토 7] │
│                                         │
│ [출처 충돌] 빵집명 성수점          D     │
│ 영업시간 · 기존 A → 표시 D               │
│ 공식 SNS와 지도 정보가 다름              │
│ 2026.06.22 확인 · 2026.06.23 재검토       │
│                    [공식 출처로 재확인 →] │
│                                         │
│ [기한 경과] 빵집명 부산점          C     │
│ 영업시간 · 기존 A → 표시 C               │
└─────────────────────────────────────────┘
```

- 충돌, 기한 경과, 14일 이내 재검토 순으로 정렬한다.
- 공식 확인으로 이동할 때 지점·항목·메뉴를 미리 선택한다.
- 모바일에서는 요약과 대기 항목을 단일 열로 배치한다.

## 9. 반응형 원칙

- 모바일 홈: 9:16 한 화면 단위 세로 스냅과 고정 하단 내비게이션
- 모바일의 목록·상세·제보: 정보 손실을 피하기 위해 일반 단일 열 스크롤 유지
- 태블릿: 지도와 목록의 40/60 분할 허용
- 데스크톱: 최대 콘텐츠 폭 1280px, 탐색은 지도/목록 동시 표시
- 상세 본문은 읽기 폭을 제한하고 우측에 저장·길찾기 보조 패널 배치 가능
- 기능과 정보 우선순위는 화면 크기에 따라 바뀌지 않음
